## Clone repository

   ```shell
   git clone https://github.com/ariniamsr/bike-sharing.git
   ```

## Instal package/library

    ```shell
    pip install -r requirements.txt
    ```

## Running

    ```shell
    cd bike-sharing/dashboard/
    streamlit run dashboard.py
    ```
